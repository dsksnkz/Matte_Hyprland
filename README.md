# ifunx - Dotfiles

## Installation Script

### Hyprland
```
sudo cd .config && git clone https://github.com/dsksnkz/ifunx-dotfiles.git && pacman -S cava rofi waybar fastfetch steam wlogout firefox nmtui helium obsidian blender logseq fish hyprlock hyprland-plugin-csgo-vulkan-fix && echo Hyprland-Default - Installation either completed or failed. You may now close this Window or report a bug. 
```

### Niri
```
sudo cd .config && git clone https://github.com/dsksnkz/ifunx-dotfiles.git && pacman -S niri-git cava rofi waybar fastfetch steam wlogout firefox nmtui helium obsidian blender logseq fish hyprlock hyprland-plugin-csgo-vulkan-fix && echo NIRI - Installation either completed or failed. You may now close this Window or report a bug.
```
